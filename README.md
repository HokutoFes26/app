このリポジトリは、第18回高専祭「北斗祭」のウェブアプリのソースコードを管理するためのものです。
本ウェブアプリでは、北斗祭での模擬店の混雑・在庫状況、イベントタイムテーブル、バスの発着時刻、落とし物情報、Q&Aをリアルタイムに更新し、スムーズにより楽しく回るためのWebアプリケーションです。  
特設ホームページについては[このリポジトリ](https://github.com/HokutoFes26/HokutoFes26.github.io)を参照してください。

## 主な機能

### 来場者向け

- **模擬店リアルタイム状況**: 全模擬店の「混雑度」と「在庫状況」を信号機形式（緑・黄・赤）で一覧表示。Supabase Realtime により、現地に行かなくても最新の状況を即座に把握できます。
- **現在地連動・周辺情報表示**: 校内に掲示された QR コードを読み取ることで、そのスポット周辺の模擬店や展示の情報を即座にピックアップして表示します。
- **インタラクティブなキャンパスマップ**: Leaflet を活用し、模擬店、展示、イベントステージの場所を直感的に探すことができます。
- **アクティビティコレクション**: 訪れたブースをチケットのように記録。集めたチケットはスタック形式のリストで振り返ることができます。
- **タイムテーブル & 進行状況表示**: ステージ企画の進行状況をバー形式で可視化。直近の予定に絞るフィルタリング機能も搭載。
- **バス時刻表**: 現在時刻から乗れるバスを優先表示します。
- **リアルタイム・ニュース配信**: 運営からの重要告知を即座に受信。最新の投稿には「New」バッジが付与されます。
- **写真付き落とし物リスト**: 忘れ物の情報を写真付きで閲覧。画像表示のオン/オフでデータ通信量にも配慮。
- **オンライン投票**: 模擬店・展示部門のリアルタイム投票。IP / LocalStorage による多重投票防止機能を備えています。
- **アプリ設定**: ライトテーマ / ダークテーマの切り替えや、日本語 / 英語の多言語に対応しています。

### 運営・出店団体向け

- **ブース管理マネージャー**: 出店担当者がスマホから数タップで「混雑度」と「在庫状況」を更新可能。
- **模擬店シフト引き継ぎ**: QRコードによるシフト交代機能。セキュアなトークンを用いてログイン状態を安全に引き継げます。
- **総合管理パネル (Admin)**: お知らせの配信、落とし物登録 (画像自動圧縮アップロード)。
- **リアルタイム集計**: 投票状況をカテゴリごとにランキング形式で即時確認。
- **サーバーステータス監視**: Supabase との接続状況やテーブルの健全性をリアルタイムにモニタリング。

### データの更新頻度

| データ名   | 更新頻度              | 存在するテーブル |
| ---------- | --------------------- | ---------------- |
| 模擬店状況 | リアルタイム(1秒以内) | stalls_status    |
| お知らせ   | リアルタイム(1秒以内) | news             |
| 落とし物   | 1.5分(DB上で変更可)   | lost_items       |
| 投票結果   | 5分                   | vote_results     |

## 内部管理と権限

アプリには 3 つのロールがあり、アクセスするパスや認証状態によって自動的に切り替わります。

| ユーザー種類   | ロール名      | 付与・識別方法                                                                 |
| :------------- | :------------ | :----------------------------------------------------------------------------- |
| **一般来場者** | `user`        | トップページにアクセスで自動付与                                               |
| **模擬店担当** | `stall-admin` | トークン付き QR コードによる認証（全体運営またはログイン済みの担当者から取得） |
| **全体運営**   | `admin`       | `/app/admin` にてログイン成功                                                  |

### 権限マトリクス

| ロール        | お知らせ | 模擬店状況 | 落とし物 |
| :------------ | :------: | :--------: | :------: |
| `user`        |   閲覧   |    閲覧    |   閲覧   |
| `stall-admin` |   閲覧   |  **更新**  |    -     |
| `admin`       | **配信** |    閲覧    | **配信** |
